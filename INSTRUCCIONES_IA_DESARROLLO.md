# 📖 INSTRUCCIONES PARA IA DE DESARROLLO

## 🎯 Tu tarea como IA de desarrollo
**NO es:** Crear un sitio bonito + agregar contenido.
**ES:** Traducir una estrategia psicológica y narrativa en código que captura atención de múltiples arquetipos, evoca sensaciones específicas, guía a una acción diferente, es técnicamente sofisticado (3D real) y accesible.

## 🏗️ ARQUITECTURA A ALTO NIVEL
```
Root Layout
├── Hero Section (Morphing + Particles)
├── About Section (Humanidad + Profesionalismo)
├── Transformation Timeline (3D interactive)
├── Perspectives (Articles/insights)
├── Skills & Expertise
├── Projects/Cases Gallery
├── Mentorship & Growth
├── Humanidad (hobbies, values)
└── Contact & CTAs
```

## 🎬 ANIMACIONES 3D — DETALLE POR SECCIÓN
- **HERO:** Partículas flotando (caos) -> scroll -> forman estructura/patrón (claridad). Tech: Three.js + GSAP.
- **TRANSFORMACIÓN/TIMELINE:** Timeline 3D interactivo (16, 17, 23, hoy).
- **PROYECTOS:**
  - *Técnico:* Cubos de datos.
  - *Educativo:* Nodos conectados.
  - *Crisis:* Líneas desordenadas → organizadas.
  - *Social:* Flujo bifurcado.

## 🎨 PALETA DINÁMICA
- CSS Variables en root que cambian por sección (Smooth transition).

## 🎭 SENSACIONES & TIMING
- Hero: "Espera... esto está bien hecho" (Rápida).
- About: "Eres humano pero capaz" (Suave).
- Transformation: "Hay camino" (Scroll-driven).
- Contact: "Vamos, hagamos algo" (Energética).

## 🚀 MVP PRIORITIZATION
**FASE 1:** Hero, About, 3-4 Case Studies, Contact.
